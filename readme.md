# Code class `fetch`

## Getting started

```sh
# clone repository
$ git clone git@github.com:voorhoede/code-class-promises.git

# go inside
$ cd code-class-fetch

# install dependencies
$ npm install

# start server http://localhost:33824
$ npm start
```
